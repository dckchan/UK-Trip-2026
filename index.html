Button Guide & Help
Control Buttons Reference
Plan Header toolbar. Shortcut to generate the full trip itinerary. Same as "Generate Full Itinerary" below. Rebuilds all 16 days, loads the map, populates Critic's Pick recommendations, and resets edit history. Use any time to start fresh.
Generate Full Itinerary Overview panel. Creates the complete day-by-day itinerary from the built-in template (16 days, all port activities, flights, hotels, cruise schedule). Also loads the interactive map with all routes and markers. Click this first when the app opens.
Download PDF Header, Overview & Modify panels. Generates a printable PDF of your itinerary and downloads it to your device. Includes the map image (except on iOS), all day schedules, and a page-numbered footer. Best for printing or sharing as a paper document.
Save Header toolbar. Saves your entire trip — including all edits, added excursions, Critic's Pick selections, and custom items — as a standalone HTML file. The saved file works offline in any browser with all your changes preserved. Filename includes the date/time (e.g. UK_Trip_2026_saved_20260915_1430.html).
Save HTML Overview & Modify panels. Same function as the Save button above. Creates a complete self-contained HTML file with your current state baked in. Open the saved file later to continue editing from where you left off — no internet required.
Save Backup Modify → Export panel. Exports your trip data as a lightweight JSON file (data only, not the full app). Useful as a safety backup — much smaller than the HTML file. Use together with "Restore" to transfer your trip data between devices or recover from errors.
Restore Modify → Export panel. Loads a previously saved JSON backup file to replace your current itinerary and selections. Useful for recovering from mistakes, transferring data between devices, or reverting to an earlier version. You'll be asked to confirm before overwriting.
Quick Tips
• Undo / Redo: Ctrl+Z to undo, Ctrl+Y or Ctrl+Shift+Z to redo. Also available via ↩ ↪ buttons on each panel. • Auto-save: Changes are automatically saved to your browser's local storage as you edit. Use "Save" for a permanent file. • Keyboard shortcuts: Ctrl+S to save to browser storage. Left/Right arrow keys navigate day tabs in the Itinerary panel. • Dark mode: Click the / icon in the header toolbar to toggle dark mode. • Font size: Use the Aa −/+ buttons in the header to adjust text size across the entire app. • Print: The Print button in Modify → Export opens your browser's print dialog. The layout auto-adjusts for paper.
